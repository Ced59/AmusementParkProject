# shared/utils

Utilitaires transverses réutilisables entre plusieurs domaines.

Les helpers communs officiels vivent ici pour éviter de disperser des petites fonctions dans chaque
future feature.

## Sous-zones actuelles

- `mapping/` : helpers génériques de mapping et pagination posés en P03 ;
- `display/` : helpers de labels d'affichage et options transverses centralisés en P09.4 ;
- `localization/` : résolution de valeurs localisées, nettoyage HTML léger et helpers de texte ;
- `angular/` : petits helpers Angular techniques réutilisables entre plusieurs écrans.

Les anciens chemins historiques ne doivent plus être réintroduits.
