# shared/models

Sous-zone destinée aux modèles transverses réutilisables entre plusieurs features.

À partir de **P03**, les contrats communs officiels vivent dans `shared/models/contracts`.
Les anciens modèles de `src/app/models/shared` restent tolérés comme zone de compatibilité,
mais les nouvelles implémentations doivent converger vers les contrats partagés.
