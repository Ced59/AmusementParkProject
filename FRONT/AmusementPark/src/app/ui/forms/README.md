# UI Forms

Composants de formulaire transverses issus de la phase M05.

- `app-ui-search-panel` orchestre uniquement l'affichage et les événements utilisateur.
- `app-ui-field-input` fournit un champ cohérent avec le design system.
- `app-ui-category-chip` sert aux filtres rapides sans dépendance métier.
- Le style est centralisé dans `src/styles/_forms.scss`.
