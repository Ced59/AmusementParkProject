# ui/layouts

Layouts applicatifs cibles de la refonte MVP.

## Layouts attendus dans les phases suivantes

- `public-app-layout` : navigation publique desktop, bottom nav mobile, footer et zone de contenu public.
- `admin-app-layout` : layout admin dense, séparé du layout public.
- `account-layout` : auth, reset password et profile.

Cette phase prépare le contrat et les styles, mais ne branche pas encore les routes sur ces layouts.
